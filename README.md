# Segametsi Cash | Loans Website

A modern, responsive fintech website built for **Segametsi Cash Loans**, a **100% Black woman-owned NCR registered credit provider** offering accessible short-term loan solutions in South Africa.

The website was designed with a **premium user experience**, combining modern UI design, smooth animations, and responsive layouts to provide visitors with a seamless way to explore services, calculate loans, and contact the company.

## Features

* Responsive **multi-page website**
* **Loan calculator** with interactive slider
* **Glassmorphism fintech UI design**
* **GSAP animations** and floating finance elements
* **Particles.js animated background**
* **Sticky shrinking navigation bar**
* **Mobile responsive hamburger menu**
* **About page with company timeline and leadership section**
* **Contact page with business details, contact form, and map**

## Built With

* HTML5
* CSS3
* JavaScript
* GSAP (animations)
* Particles.js

## Purpose

This project showcases a **premium modern fintech-style website** built with clean front-end technologies to represent the services and brand identity of Segametsi Cash Loans.
